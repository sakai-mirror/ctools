Importing material from another site you own

Note: To complete the procedures in this document, you must be assigned a role having the necessary permissions. To determine your role, follow the directions in Participant roles. For a basic understanding of roles and permissions, see Permissions and roles: Overview.

Note: If you are planning to combine rosters, it's best to do so before adding content. Combining rosters does not import content (e.g., calendar items, assignments, forums, announcements) into the parent site. If you already have content in the sites you want to combine, you should be able to import instructor-generated content (e.g., announcements, assignments, tool settings) using the instructions below. However, submitted material, such as completed assignments, forum postings, or grades, cannot be imported using the following method. You may want to download and save any submitted material before combining rosters. Once rosters have been combined into a parent site, the tools for the original sites are disabled. However, you can enable them by using Worksite Setup from My Workspace. For instructions, see Adding, editing, or deleting tools on the menubar from Worksite Setup.

For any course or project site you create, you can import content from other sites you own. You can also combine material from more than one site. To do this:

  1. In the menubar of the destination site, click Site Info.

  2. At the top of the Site Info screen, click Import from Site.

  3. Select the box next to the site that contains the source material. You can choose more than one site. Then click Continue.

  4. Under the name of the site(s) you selected, check the box next to each tool that contains source material you want to import. If a box is grayed out, it means the corresponding tool is not available in that site. Then click Finish.

Note: Items saved as drafts in the source site will be imported as drafts, which means they are not immediately viewable by participants. To make them viewable, you must post them. For instructions, see Posting an assignment previously saved as a draft.
